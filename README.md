# C++ How to Program — Fourth Edition

This repository contains my solutions to exercises from the book  
**C++ How to Program, Fourth Edition** by Deitel & Deitel.

## About

The purpose of this repository is to practice and improve my understanding of:

- C++ fundamentals
- Structured programming
- Object-oriented programming
- Functions and recursion
- Arrays and strings
- Pointers and references
- Classes and objects
- File handling
- Data structures
- And other C++ concepts

## Repository Structure

Each chapter contains exercise solutions and example programs related to the topics covered in the book.
